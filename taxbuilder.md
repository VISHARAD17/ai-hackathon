  Tax Filing AI Agent
  
  The Flow:
  
  "Help me file my 2025 taxes"
    ↓
  AI generates:
    → Income source selector (W2, 1099, freelance, investments)
    → Document upload zones for each source
  
  User uploads W2
    ↓
  AI generates:
    → Extracted data shown in cards
    → "Looks good" or "Missing info" badges
    → Additional income questions
  
  User answers questions
    ↓
  AI generates:
    → Deduction finder (asks smart questions)
    → "Did you work from home?" → home office deduction calculator appears
    → "Any charitable donations?" → donation tracker appears
    → "Student loans?" → interest deduction calculator appears
  
  AI finds all deductions
    ↓
  AI generates:
    → Tax summary dashboard
    → Federal vs State breakdown
    → Refund/owed amount (big number, animated)
    → Comparison: "You're getting $2,340 more than if you used standard deduction"
  
  User reviews
    ↓
  AI generates:
    → Form preview (actual 1040 form filled out)
    → Line-by-line explanation (click any line → AI explains it)
    → E-file button
  
  User clicks e-file
    ↓
  AI generates:
    → Signature pad
    → Confirmation screen
    → "Your refund will arrive in 2-3 weeks"
    → Calendar reminder generator
  
  Generative UI moments:
  
  - Upload W2 → AI generates income summary card
  - Answer "yes" to home office → calculator appears
  - Each deduction question → new form appears
  - Review → actual tax form generates
  - Click any line on form → explanation popup generates
  
  Pros:
  
  - Everyone relates — everyone hates taxes
  - Clear before/after — "You owe $5k" vs "You get $2k refund"
  - Deduction finder is magic — AI asks questions you didn't know to ask
  - Form generation is impressive — actual IRS forms filled out
  - Easy to demo — use sample W2, show the full flow
  - High value — people pay $200+ for TurboTax
