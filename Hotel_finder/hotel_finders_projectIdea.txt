## Problem Statement## The Fragmentation Problem
The modern travel planning ecosystem is highly fragmented. Accommodation options are scattered across walled gardens with entirely different inventory types, pricing structures, and terminologies (e.g., OYO hotels versus Airbnb private rentals). Travelers must manually open 5 to 10 browser tabs, continually re-enter dates, and track variations in guest capacities across disparate user interfaces.
## The "Hidden Fees" and Dynamic Pricing Trap
Standard aggregate platforms often mask the true cost of lodging. The final checkout screen frequently reveals mandatory, non-negotiable costs like cleaning fees, service surcharges, local occupancy taxes, and resort fees. Furthermore, booking sites track individual user cookies, triggering artificial price surges based on perceived urgency or repeated search queries.
## The Cognitive Load of Unstructured Reviews
Traditional reservation websites present hundreds of unorganized, text-heavy reviews. Consumers struggle to filter out fake or compensated reviews and must read through pages of feedback to find critical, niche preferences—such as stable Wi-Fi speeds for remote work, pet friendliness, or neighborhood noise levels at night.

## Parallel Multi-Agent Swarm Integration
Instead of relying on a single linear script, the system orchestrates a swarm of distinct AI agents working concurrently. Upon receiving a natural language request, the Orchestrator Agent spawns multiple platform-specific Scraper Agents. These agents deploy serverless, headless browsers to extract markdown data directly from providers simultaneously.
## Dual-Layer Isolated Memory Architecture
Each specialized agent maintains its own short-term memory partition during execution. This allows them to manage targeted sub-tasks independently—such as navigating anti-bot challenges or tracking multi-page results—without causing cross-agent data corruption. Once extraction completes, agents stream their findings into a shared Central Blackboard Memory.
## Intelligent Consolidation & Analysis
A centralized Synthesis Agent evaluates the aggregated memory data. It eliminates duplicate listings, isolates all hidden line-item fees to calculate the true final cost, analyzes text sentiment for specific user requirements, and formats the consolidated data into a clean, actionable overview.
## Dynamic Long-Term Personalisation
The platform features an evolving user memory layer that retains individual preferences over time. By learning from past queries and interactions, the AI dynamically tailors all future searches, highlighting options that match the traveler's historical habits without requiring manual filters.
